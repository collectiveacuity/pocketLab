# Roadmap

## Features
- Docker Wrapper
- GitHub / BitBucket / Gitlab Repos
- OS Independence
- AWS Deployment Management
- Let's Encrypt SSL Certificates
- LocalTunnel.me / Ngrok
- PingAPI / Uptime Robot Monitoring
- Test Sequencing
